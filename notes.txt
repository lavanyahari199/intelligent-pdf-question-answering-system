Phase 1 - Project Setup (Today)

Create this folder:

C:\Portfolio Projects\intelligent-pdf-question-answering-system

Inside create:

intelligent-pdf-question-answering-system
│
├── README.md
├── app.py
├── requirements.txt
│
├── data
├── screenshots
├── documents
└── notebooks

Don't worry about contents yet.

Phase 2 - Create Virtual Environment

Open terminal inside project folder.

Run:

python -m venv venv

Activate:

venv\Scripts\activate

You should see:

(venv)

at the beginning of the terminal line.

Phase 3 - Install Packages

Run:

pip install streamlit
pip install PyPDF2
pip install sentence-transformers
pip install faiss-cpu
pip install google-generativeai
pip install numpy pandas scikit-learn nltk
Phase 4 - Generate requirements.txt

Run:

pip freeze > requirements.txt

This creates the dependency file recruiters expect.

Phase 5 - First Git Commit

Before writing code:

git init
git add .
git commit -m "Project setup"

We are going to do Git properly from day 1.
